# State Set
## Definition
A State Set is a set of terms which a [State Machine](./StateMachine.md) may use to describe the current condition of a [Device](./Device.md).

<br>
<br>
<br>

## Example
The State Set listed for the *Plymouth IoT Conveyor Belt* included
- On
- Off
- Running
- Paused
- Overheated
